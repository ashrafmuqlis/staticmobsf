# staticmobsf
Mobile Security Framework (MobSF) static analysis server How to's etc.

## Navigate to MOBSF folder
```
cd Mobile-Security-Framework-MobSF
```

## Start MOBSF server
```
sudo bash ./run.sh
```

## Test MOBSF front end
```
curl http://localhost:8000
```

## Check <server-ip>
```
hostname -I
```


## Open MOBSF browser
```
http://<server-ip>:8000
```
